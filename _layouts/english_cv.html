PrintWizard - Immediate tracing for smoother debugging and code exploration
Spring 2024
Master thesis done at SYSTEMF, EPFL.

Designed PrintWizard, a history debugger. PrintWizard records the complete history of a Java program and then lets you interactively explore what happened. It helps you debug problems that are difficult to solve with a standard debugger.

The project consists of two parts. A Java backend, a plugin for the Java compiler that modifies the program at compile time. Then, when executed, the program records its complete history in an event log. And a graphical user interface implemented in Javascript and HTML that allows interactive exploration of the program's history.
